# 🎵 تطبيق استماع القرآن الكريم

تطبيق ويب حديث وجميل للاستماع إلى القرآن الكريم بأجمل أصوات القراء

## ✨ الميزات الرئيسية

### 📖 القراء المدعومون
- **احمد العجمي** 🟢 (أخضر بارد)
- **وديع حمادي اليمني** 🔵 (أزرق فاتح)
- **احمد النفيس** 🔴 (أحمر)
- **خالد الجليل** 🟣 (بنفسجي)

### 🎧 مميزات المشغل
- ✅ مشغل صوت متقدم وسلس
- ✅ تحكم كامل بمستوى الصوت
- ✅ شريط تقدم تفاعلي مع دائرة متحركة
- ✅ عرض الوقت الحالي والمدة الإجمالية
- ✅ أزرار تحكم سهلة الاستخدام

### 🔄 وضع التكرار
- بدون تكرار
- تكرار السورة مرة واحدة
- تكرار الكل (السور اللاحقة)

### 🔁 التشغيل التلقائي
- تشغيل تلقائي للسورة التالية
- متوقف بشكل افتراضي

### ⬇️ التحميل
- تحميل السورة بصيغة MP3
- أسماء ملفات واضحة تتضمن السورة والقارئ

### 🔍 البحث والفلترة
- بحث سريع عن السور
- البحث بالاسم العربي أو الإنجليزي أو الرقم

### 🎨 التصميم
- تصميم حديث وجميل
- ألوان مميزة لكل قارئ
- واجهة سهلة الاستخدام
- متوافق مع جميع الأجهزة (responsive)

### 🔗 المشاركة
- زر لنسخ رابط التطبيق بسهولة
- مشاركة التطبيق مع الآخرين

## 📋 قائمة السور

يتضمن التطبيق كل السور الـ 114 من القرآن الكريم

## 🎯 كيفية الاستخدام

1. **اختر القارئ**: انقر على صورة وشاشة القارئ المفضل
2. **اختر السورة**: اختر السورة من القائمة أو ابحث عنها
3. **شغّل التلاوة**: اضغط على زر التشغيل
4. **تحكم بالتشغيل**: استخدم الأزرار والمنزلقات للتحكم

## 🛠️ التكنولوجيا المستخدمة

- HTML5
- CSS3 (مع تصميم متجاوب)
- JavaScript (Vanilla)

## 📱 المتطلبات

- متصفح حديث يدعم HTML5 Audio
- اتصال إنترنت
